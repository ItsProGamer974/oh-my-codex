## ⚙️ Main features

OmX is built around a few core tools that help you work with code support in one place.

### 🪝 Hooks

Hooks let you connect actions to events. Use them when you want the app to react to changes in your workflow.

You may use hooks to:

- Start a task
- Track changes
- Trigger an update
- Send work to another tool

### 👥 Agent teams

Agent teams let you group helpers by task. This makes it easier to split work across different parts of a project.

You may use agent teams to:

- Plan work
- Review code
- Handle repeated tasks
- Keep roles separate

### 🖥️ HUDs

HUDs give you quick on-screen views of important info. They help you stay focused without opening extra windows.

You may use HUDs to view:

- Task status
- Active agents
- Recent actions
- Live changes

### 🧰 Workflow tools

OmX also includes other tools that help you manage your setup. These tools are made to keep your work organized and easier to follow.

## 🛠️ Troubleshooting

If the app does not open, try these steps:

- Check that the file finished downloading

- Right-click the app and choose Run as administrator

- Restart your PC and try again
